# RyuuseiManager
An experimental save data manager for Mega Man Star Force Legacy Collection.

## Prerequisites
You must have purchased and you're playing this game on Steam. This program will not work without the game play records.

## Usage
- Make sure you have any proper game records on Steam. If you can't see the game you're trying to manage, please run the game you'd like to manage, and save at least once, then open this program again.
- To share a save data file you're currently playing, click "Export" button, it will create a data0XXSlot.bin at directory you exported.
- To import a save data file, click "Import" button and choose the save data file exported with this program. Do not import save data from Steam userdata directory, or the save data file extracted from Nintendo Switch.
